# Memoline

**発展式メモツール** — メモを書き留め、育て、テーマへと結晶させる3画面構成のローカルWebアプリ。

---

## コンセプト

Memolineは「メモを育てる」ことを目的に設計されています。

短い断片（メモ）を捕捉し、エディタで展開・接続し、浮かび上がるテーマをメタラインに結晶させる。この3段階の流れを1つのツールで完結させます。データはすべてブラウザのローカルストレージに保存され、サーバーもアカウントも不要です。

DEMO:https://rashita.github.io/memologue/

---

## 3つの画面

### 1. メモライン
短い思考や気づきを書き留めるタイムライン。広めのマージンで余白を意識したレイアウト。メモをクリックするとノートラインのエディタ先頭に送り込めます。

### 2. ノートライン
3カラム構成の作業画面。

| 左パネル | 中央 | 右パネル |
|----------|------|----------|
| メモライン（参照用） | エディタ | メタライン |

- **編集モード**: テキストエリアで自由に書く
- **プレビューモード**: Markdownの見出しを認識してレンダリング。見出し単位でセクションの折り畳みが可能
- 左右パネルはそれぞれ独立してトグル開閉できる

### 3. メタライン
エディタに書かれた `#ハッシュタグ` を自動抽出して一覧表示。タグをクリックするとノートラインのプレビューに飛び、そのタグを含むセクションだけを表示してフィルタリング。

## 使い方

### インストール

```
git clone https://github.com/rashita/memologue.git
```

`index.html` をブラウザで開くだけで動作します。ビルド不要、依存パッケージなし。

### 基本操作

| 操作 | 方法 |
|------|------|
| 画面を移動する | 左右端の ‹ › ボタン、または `Alt + ←` / `Alt + →` |
| メモを保存する | `Enter`（改行は `Shift + Enter`） |
| メモをエディタに送る | メモラインまたは左パネルのメモをクリック |
| 見出しを折り畳む | プレビューモードで見出し行をクリック |
| タグでフィルタする | メタラインのタグをクリック |
| フィルターを解除する | チップの ✕ ボタン、または `Escape` |

### Markdownの記法

エディタのプレビューモードで以下の記法が有効です。

```
# 見出し1
## 見出し2
### 見出し3

**太字**  *イタリック*

#ハッシュタグ（スペースなしで # を付ける）
```

## データについて

- ブラウザの `localStorage` に保存（`ml_memos` / `ml_note`）
- サーバー送信なし、完全ローカル動作
- ブラウザのキャッシュクリアやシークレットモードではデータが消えます

---

## 技術構成

- HTML / CSS / JavaScript（フレームワーク・ライブラリなし）
- 単一ファイル構成（`index.html`）
- ブラウザ標準APIのみ使用
